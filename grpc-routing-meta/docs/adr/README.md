# Architecture Decision Records

The *why* behind the hard-to-reverse choices. An ADR is recorded only when the decision
is hard to reverse, surprising without context, and the result of a real trade-off.
Once **Accepted** an ADR is immutable — supersede it with a new one rather than editing.

Format: **Status · Context · Decision · Consequences**.
